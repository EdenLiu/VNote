设计一款小工具VNote， 是一款主打“简单但够用”的轻量级任务管理工具，其核心功能围绕快速记录、智能规划
UI可以沿用微软todo list这个软件来设计

以下按照模块化的结构整理的需求清单：：
1.智能规划与建议
- 我的一天：一个每日清零的个性化看板，用户可以在此聚焦当日最重要的任务 。
- 建议功能：系统会自动从其他任务列表中推荐今日可执行的任务，帮助用户规划 （支持在列表设置中开启/关闭此功能 ）。
- 智能列表：自动分类，包括“计划内”（有截止日期的任务）、“重要”（已标记星标的任务）、“已标记电子邮件”以及“已完成”视图 。
2. 任务编辑与组织
- 快速添加：支持快速输入任务描述，操作轻量化 。
- 子任务 (步骤)：支持将复杂任务拆解为多个步骤 。
- 标签与分类：
- 标签：支持通过 #话题 进行非正式分组 。
- 类别：支持使用Outlook内置的“类别”（如红、黄、绿色标签）进行正式分类，并可与Outlook同步 。
- 备注与附件：支持添加富文本备注，并支持上传文件（单个文件限制 25 MB）。
- 优先级标记：支持通过“星标”标记重要任务 。
 3. 时间与提醒管理
- 截止日期与提醒：支持设置具体的截止日期和独立于截止日期的提醒时间 。
- 重复周期：支持设置重复任务（如：每天、每周、每月）。

按照这个需求来实现工具，先提出实现框架，然后再开始实现代码，按需求合理将任务分配给subagent执行
架构：Electron + Vite + React + TypeScript 

构建一个Agentic应用程序，使用的底层 Harness 与 Claude Code 相同，主要用来抓取整理分析照明系统领域的功能设计，最终输出以下内容：
- 当前主要照明系统对当前功能的设计方案
- 分析当前主要照明系统对于该功能的方案
